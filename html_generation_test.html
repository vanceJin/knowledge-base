HTML生成测试

1. 静态HTML创建

静态HTML是直接编写的HTML文件，不需要任何动态生成过程。

<!DOCTYPE html> <html> <head> <title>静态HTML示例</title> </head> <body> <h1>Hello, World!</h1> <p>这是一个静态HTML页面。</p> </body> </html>

2. 服务器端渲染 (SSR)

服务器端渲染是在服务器上生成HTML，然后将完整的HTML发送到客户端。

// 示例：使用Node.js和Express进行服务器端渲染 const express = require('express'); const app = express(); app.get('/', (req, res) => { const html = ` <!DOCTYPE html> <html> <head> <title>服务器端渲染示例</title> </head> <body> <h1>Hello, ${req.query.name || 'World'}!</h1> <p>当前时间: ${new Date().toLocaleString()}</p> </body> </html> `; res.send(html); }); app.listen(3000, () => { console.log('服务器运行在 http://localhost:3000'); });

3. 客户端渲染 (CSR)

客户端渲染是在浏览器中使用JavaScript生成HTML。

// 示例：使用JavaScript进行客户端渲染 document.addEventListener('DOMContentLoaded', () => { const container = document.getElementById('content'); const html = ` <h1>Hello, World!</h1> <p>当前时间: ${new Date().toLocaleString()}</p> `; container.innerHTML = html; });

4. 基于模板的生成

基于模板的生成使用模板引擎（如Handlebars、EJS、Pug等）来生成HTML。

// 示例：使用Handlebars模板引擎 const Handlebars = require('handlebars'); const template = Handlebars.compile(` <!DOCTYPE html> <html> <head> <title>{{title}}</title> </head> <body> <h1>{{heading}}</h1> <ul> {{#each items}} <li>{{this}}</li> {{/each}} </ul> </body> </html> `); const html = template({ title: '模板示例', heading: 'Hello, World!', items: ['Item 1', 'Item 2', 'Item 3'] }); console.log(html);
