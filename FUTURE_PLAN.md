# Jekyll 知识库后续优化计划

根据样式审查报告和设计指南，制定以下后续优化计划。

## 🎯 核心优化方向

### 1. 导航系统完善
- **实现面包屑导航**：根据设计指南，添加面包屑导航，最多显示 4 级层级
- **优化侧边栏**：实现 2 级导航，添加图标增强识别
- **顶部导航**：添加 5-7 个主要菜单项，保持简洁命名

### 2. 搜索功能实现
- **集成 Lunr.js**：添加全文搜索功能
- **搜索优化**：
  - 索引字段：title (boost: 10), content, tags
  - 支持模糊搜索
  - 实时搜索建议
  - 搜索结果高亮
- **性能优化**：使用 `jekyll-lunr-js-search` 插件，生成 `search.json` 索引文件

### 3. 性能优化
- **构建优化**：
  - 启用增量构建 (`incremental: true`)
  - 关闭相似性搜索 (`lsi: false`)
  - 不生成未来日期文章 (`future: false`)
  - 不生成草稿 (`drafts: false`)
- **资源优化**：
  - 压缩和合并 CSS/JS
  - 异步加载
  - 响应式图片 (`srcset`)
  - 懒加载
- **缓存策略**：
  - 浏览器缓存
  - CDN 缓存静态资源

### 4. SEO 优化
- **元数据标签**：
  - 页面标题
  - 描述
  - 关键词
  - Open Graph 标签
  - Twitter Card 标签
- **结构化数据**：添加文章结构化数据
- **站点地图**：使用 `jekyll-sitemap` 插件自动生成

### 5. 功能增强
- **相关文章推荐**：基于标签或分类显示 3-5 篇相关文章
- **深色模式切换**：添加手动切换按钮
- **代码高亮**：优化代码块样式
- **表格样式**：添加响应式表格样式
- **图标系统**：集成 Font Awesome 或 Material Icons

## 📁 文件结构优化

### 1. 新增文件
- `_data/navigation.yml`：导航配置
- `_includes/breadcrumb.html`：面包屑导航
- `_includes/search.html`：搜索功能
- `_includes/related-posts.html`：相关文章
- `assets/js/search.js`：搜索脚本
- `assets/js/theme-switcher.js`：主题切换

### 2. 修改文件
- `_config.yml`：添加插件和配置
- `_layouts/default.html`：集成新组件
- `_includes/head_custom.html`：添加必要的脚本
